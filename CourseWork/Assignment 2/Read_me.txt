
Pre-requisites for running the program
--------------------------------------
1. Python 3.5 or above installed
2. Python libraries sklearn, pandas, numpy installed.

How to run the program for evaluation
-------------------------------------

1. Unzip the file to any location
2. Replace the 'input.csv' file with the file containing test data for evaluation
3. Open Terminal in the location where files are saved
4. Run the script by typing 'python prediction.py'. Output classes in each case will be displayed in each line


Note: Lines 32 - 36 of sourcecode 'prediction.py' may be modified for changing output format
